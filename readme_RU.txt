== Installation ==

Войдите в админ-панель WordPress, перейдите в меню «Плагины» и нажмите «Добавить». В поле поиска введите «stepFORM» и нажмите «Поиск плагинов». После того, как вы найдете плагин, нажмите «Установить сейчас».

После активации, посетите раздел ‘Настройки > stepFORM’ и привяжите аккаунт. После привязки следуйте инструкции на странице.

== Description ==

**Этот плагин даёт возможность автоматически встроить форму из stepFORM — достаточно разрешить доступ и выбрать форму из списка.**

[stepFORM](https://stepform.io/ru) — это конструктор для создания форм, квизов и различных онлайн опросов. С его помощью вы сможете собирать контактные данные клиентов, проводить пошаговые опросы, предлагать автоматический расчет стоимости услуг или товаров. Собрать форму происходит в визуальном режиме, без специальных навыков и трат на поддержку.

Разработка в конструкторе выполняется очень быстро и просто, для этого достаточно выбрать, добавить и настроить необходимые элементы, после чего использовать получившуюся форму и получать с неё потенциальные лиды!

#### 18 встроенных элементов для размещения внутри конструктора
Внутри конструктора уже предусмотрены все необходимые элементы для создания самых разных форм. Размещайте внутри формы элементы для сбора информации, сложные формы с множеством расчетов и даже корзину с возможностью добавления нескольких товаров и последующей оплатой. Полный список элементов для использования:

[Множественный выбор](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_1) — предоставляет возможность пользователю выбрать несколько пунктов в нем. Настройки позволяют указать минимальное и максимальное количество пунктов для выбора. Отличном подойдет для выбора дополнительных услуг, опций, товаров и т.п.

[Одиночный выбор](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_2) — чаще всего используется для выбора основной услуги или только одного варианта из нескольких представленных.

[Выбор изображений](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_3) — позволяет пользователю выбрать вариант ответа с прикрепленным изображением. По умолчанию можно выбрать только один вариант ответа, но при необходимости

[Выпадающий список](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_4) — предназначен для выбора одного пункта из множества. В отличие от одиночного выбора позволяет скрыть множество элементов, что делает его более компактным.

[Диапазон](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_5) — позволяет выбрать пользователю число с помощью ползунка из заданного диапазона. Его настройки позволяют выбрать числовой отрезок и назначить шаг.

[Корзина](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_6) — данный виджет уникален и позволяет создавать мини-магазины при помощи сбора различных конфигураций внутри формы с дальнейшей отправкой уведомлений на почту, telegram или в CRM систему и даже получением оплаты.

[Формула](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_7) — предназначен для создания расчёта на основе выбранных параметров в форме. Поддерживает логические выражения, а также более 20 математических функций!

[Текстовый ввод](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_8) — обычное поле для ввода текста. Может служить для ввода небольшого текста. Также позволяет настроить маску ввода для облегчения ввода пользователю.

[Цифровой ввод](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_9) — поле для ввода числа, которое в дальнейшем может участвовать в расчетах ваших формул.

[Электронная почта](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_10) — поле предназначено для ввода email адреса. Настройки валидации уже заданы технически и задать неверный email адрес не выйдет.

[Мобильный телефон](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_11) — как и поле для ввода почты имеет уже заданные технические настройки валидации. Поддерживает валидацию более 10 стран.

[Дата и время](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_12) — поля для выбора для даты в виде календаря и времени. При необходимости время может быть отключено.

[Загрузка файлов](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_13) — предоставляет возможность пользователям загружать файлы разных форматов. В элементе предусмотрена тонкая настройка форматов загружаемых файлов. Вы можете запретить или наоборот разрешить загрузку определенных форматов файлов. 

[Рейтинг](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_14) — используется для сбора обратной связи путем выбора количества звезд.

[Кнопка](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_15) — позволяет разместить кнопку с назначаемым действием. Предусмотрена печать формы, отправка формы, переход на страницу оплаты или открытие ссылки при нажатии на нее.

[Изображение](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_16) — в данный элемент вы можете разместить собственное изображение формата jpg, jpeg, png, gif и svg.

[Сообщение](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_17) — является обычным текстом с возможностью стилизации (жирность, размер, цвет и т.п).

[Код](https://stepform.io/ru/docs/elementy-i-ikh-nastroyki#anchor_18) — предоставляет возможность размещения собственного кода: html, css, javascript.


#### Задайте дизайн подходящий вашему сайту или бренду
Используйте готовые цветовые схемы или задайте собственную при помощи множества настроек дизайна. Вы можете изменить дизайн каждого элемента размещенного в форме. Недостаточно встроенных настроек дизайна 🤔? Вы всегда можете добавить собственные CSS стили прямо в конструктор.

#### Принимайте платежи внутри формы
Настройте интеграцию с одной из шести возможных платежных систем и получайте платежи после заполнения форм. Хотите сохранять ответы только после успешной оплаты? С помощью юKassa вы можете сохранять информацию после заполнения только в случае успешной оплаты со стороны пользователя.

#### Получайте ответы во внутреннюю CRM
Все ответы попадают в удобную внутреннюю CRM систему. В ней есть возможность выгружать полученные ответы, а также создавать собственные статусы и проставлять по мере их обработки. Не хотите использовать нашу CRM? Подключите интеграцию с Bitrix24 и получайте их автоматически после заполнения ваших форм.

#### Получайте уведомления о заполнении форм
Вы можете подключить к форме уведомление на почту, через вебхуки, СМС сообщения на ваши номера телефона, уведомление в личные сообщения Telegram или группу, а также возможность сохранять их внутри Firebase. Выберите для вас удобный способ получения уведомлений и не пропускайте ни одну заявку!

#### Используйте библиотеку готовых шаблонов
В ней представлено больше 100 разных шаблонов на разную тематику. Начиная от простых форм сбора обратной связи заканчивая сложными калькуляторами с множественным количеством формул. Взять готовый шаблон или использовать его за основу? Решать только вам.

#### Формы адаптированы под все виды устройств
После создания формы она автоматически адаптируется под любые экраны устройств. Вам не нужно продумывать отдельно мобильные или планшетные версии форм, конструктор сделает все за вас. Вы можете сразу посмотреть как будет выглядеть созданная вами форма на разных устройствах прямо внутри конструктора.

#### Собирайте статистику по посещениям форм
К формам можно подключить интеграцию с сервисами аналитики. На выбор доступно 5 сервисов, который помогут вам проводить аналитику по посещению форм и достижений уже предусмотренных целей/событий внутри конструктора. Начиная от открытия формы и заканчивая заполнением конкретных форм, а далее и отправкой самой формы.

## FAQ

= Можно ли пользоваться сервисом бесплатно?=
Да, при регистрации в сервисе предоставляется семи дневный пробный период с полным функционалом. После окончания данного периода на ваш аккаунт будет назначен бесплатный тариф “Нулевой”.

= Как отключить надпись о мошенниках?=
Все надписи не касающиеся формы будут отключены автоматически после оплаты любого тарифа.

= Надо обновлять код формы на сайте после внесения изменений в форму?=
Нет, он обновляется автоматически.

= Предоставляются ли скидки при оплате тарифов?=
Мы предоставляем 30% при оплате любого тарифа на год.

= Возможно ли совершить оплату по выставленному счету?=
Такая возможность есть. Счет предоставляется на основе договора оферты. Получить его можно при обращении в техническую поддержку с предоставлением данных статьи.

= Можно ли использовать форму по ссылке?=
Да, при создании формы генерируется прямая ссылка. Вы можете ее использовать для отправки вашим пользователям. Например, для оформления заказа.

= Куда можно отправлять полученные ответы с формы?=
Ответы с формы можно отправлять на почту, на телефон через sms или telegram, в CRM.

= С помощью вашего сервиса можно сделать квиз?=
Конечно! На каждом тарифе есть возможность создания страниц в формах. Одна страница будет являться одним шагом в вашем квизе.

= Что за интеграция с социальной сетью ВК?=
Данная интеграция позволяет разместить форму в вашей группе через приложение. В настройках можно выставить получение уведомлений о заполнении форм администраторам через личные сообщения.
